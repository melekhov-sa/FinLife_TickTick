## Дашборд

### Прогноз баланса на конец месяца
На странице бюджета: текущий суммарный баланс ± все плановые операции до конца месяца.
Сразу видно, уйдёт ли в минус. Данные уже есть в `upcoming_payments`.
После того как отладим расчёт на странице бюджета — перенести виджет на дашборд.

### Виджет "Неделя"
Мини-блок под "Сегодня": % задач выполнен за текущую неделю, серии привычек, общий расход.
Данные частично уже есть в `EfficiencySnapshot`.

### Deadline-риск
Если задача запланирована на послезавтра, а последние несколько дней ты ничего не выполнял —
мягкое визуальное предупреждение рядом с задачей (без кнопок, только маркер).

## Задачи и план

### Шаблоны наборов задач
Сохранить набор задач как шаблон (например "Подготовка к поездке", "Уборка", "Конец месяца").
Одна кнопка — создаёт все задачи из шаблона сразу с нужными датами относительно старта.
Хранится как именованный список задач с относительными смещениями дней.

### Задачи без даты — еженедельный напоминатель
У пользователя копятся задачи без `due_date` и про них забывают.
Раз в неделю push: "У тебя N задач без даты — хочешь разобраться?"
На странице плана — отдельная секция "Без даты" в конце.

### Умный перепланировщик после перерыва
Когда просроченных задач 5+, на дашборде появляется кнопка "Перепланировать".
Равномерно раскладывает просроченные задачи по ближайшим свободным дням.
Особенно полезно после отпуска или болезни.

## Финансы

### Анализ трат по категориям (month-over-month)
На странице транзакций: "В этом месяце на кафе +40% к среднему за 3 месяца."
Нужна агрегация `amount` по `category_id` с историческим сравнением.

### Предупреждение о лимите при создании операции
Лимиты по категориям уже есть в бюджете. Но при добавлении транзакции — показывать
предупреждение прямо в форме: "Эта операция превысит лимит на Кафе (осталось 200₽)" или
"Лимит на Коллекция уже превышен на 500₽". Не блокировать — только информировать.

### Долги и займы
Отдельный трекер: "я должен" и "мне должны". Сумма, кому/от кого, дата, комментарий.
Push-напоминание ближе к ожидаемой дате возврата.
Итоговое сальдо: на сколько ты в плюсе или минусе по долгам.

### Сканирование чека (OCR)
Сфотографировал чек прямо из формы создания транзакции → OCR распознаёт сумму и по
возможности категорию → поля заполняются автоматически. Можно использовать Google ML Kit
или Tesseract через backend.

## Привычки

### Счётчик-привычка (N раз в день)
Тип привычки "выполнить X раз за день" — например "выпить воду 8 раз" или "отжаться 3 подхода".
Кнопка выполнения нажимается N раз, прогресс виден (3/8 💧). Привычка закрывается только
когда достигнут дневной счётчик. Streak считается если за день набрал норму.

## Коллекционирование и инвестиции

### Модуль коллекций
Отдельный раздел для учёта коллекций: купюры, монеты, марки, карточки Покемон (и любые другие).
Каждый предмет: название, год/серия, страна, состояние (VF/EF/UNC/...), цена покупки,
оценочная стоимость, фото, дата приобретения.
- Суммарная оценочная стоимость коллекции с динамикой
- Фильтрация по типу коллекции, стране, году
- Want list — список предметов которые хочешь найти, с бюджетом
- Расходы на пополнение автоматически тянутся из транзакций по категории

### Ценовые алерты и портфель инвестиций
Два сценария в одном модуле:
1. **Коллекция** — задать целевую цену предмета (карточка Покемон, монета), push когда рынок
   достиг порога. Интеграция с TCGPlayer API (Покемон) или аналогами для нумизматики.
2. **Фондовый рынок** — учёт акций и облигаций: тикер, количество, цена покупки.
   Текущая стоимость портфеля, P&L, алерт когда цена акции достигла целевой.
   Источник котировок: MOEX API (для российских бумаг) или Yahoo Finance.

### Курсы валют на дашборде
Мини-блок с актуальными курсами основных валют (USD, EUR, CNY).
Полезно для оценки иностранных предметов коллекции и валютных активов.
Источник: ЦБ РФ API (бесплатный, официальный).

## Мотивация и геймификация

### Расширенный дайджест с шарингом
К существующему еженедельному дайджесту добавить ключевые цифры: задачи выполнено,
лучший habit streak, потрачено/заработано за месяц, XP набрано, уровень.
Кнопка "Поделиться" — генерирует красивую карточку-картинку (как Spotify Wrapped),
которую можно скинуть друзьям.

### Архив достижений / рекорды
Отдельная страница или секция в профиле:
- Лучший streak по каждой привычке (ever)
- Рекордная неделя продуктивности
- Хронология всех level-up событий
Данные частично уже есть в `XpEvent` и `HabitOccurrence`.

### Паттерн продуктивности
"Ты лучше всего работаешь в среду утром" — на основе `completed_at` за последние 30 дней.
Heatmap по дням недели × часам суток. Страница эффективности.

## Интеграции

### Telegram-бот
Быстрое добавление задач и операций через Telegram (текст → парсинг).
Утренний и вечерний дайджесты туда же.
Высокий приоритет — мобильный use case без открытия браузера.

### Web Share Target (мобильный)
При share ссылки/текста из браузера на телефоне — создаётся задача с заметкой.
Нужно обновить `service-worker.js` + добавить `share_target` в `manifest.json`.
